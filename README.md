# Plataforma de Alquiler Vacacional - Proyecto Bootcamp

##  1. ¿Qué Problema Resuelve?

Este sistema ayuda a las plataformas de alquiler vacacional a gestionar de manera eficaz la verificacion de las licencias,los limites de dias de alquiler ,registre incidencias y facilite la fiscalización, beneficiando a turistas, anfitriones, vecinos y autoridades.Es importante porque esto permite tener mayor control sobre la regulación de documentacion.

---

## 2. Usuarios Principales 

1. **ANFITRIÓN**:Crear y gestionar los anuncios y avisos, aceptar o rechazar solicitudes de huespedes, garantizar que el alojamiento sea seguro y legal.
2. **HUESPEDES**:Registrarse y crear perfil, buscar y reservar el alojamiento,dejar reseñas y calificiones del lugar,cumplir con las normas.
